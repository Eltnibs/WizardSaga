 Introduction: The Game starts with a wizard that goes through the magical door, and going from door to door

 Documentations of Your Game: The game is a platformer game where you need to get to the end while navigating through the obstacles and get to the exit door, the game have a checkpoint system where you will revive if you died.

 Manual / Instructions: 
  - move = WASD
  - jump = spacebar
  - pause game = ESC / P

> Hazards:
  - spikes
  - spinning pillar
  - bounce pad
  - water
  - moving platform
  - breakable box
  - interactable door (open when near/close when far)

> features
  - checkpoint
  - character animation (idle/run/jump)
  - coin widget
  - pause menu
  - 3 levels of platformer / 2 being main menu and winning menu
  - swaying cloud
